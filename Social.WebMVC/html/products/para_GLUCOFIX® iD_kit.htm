GLUCOFIX® iD kit

Domaines d’utilisation : Lecteurs de glycémie, Lecteurs de glycémie

Présentation, Propriétés

Le lecteur de glycémie GlucoFix iD est destiné à l'autosurveillance du diabète. Sans calibration, léger (39 g), ne nécessitant qu'une microgoutte de sang (0,5 μl).

Résultat exprimé en mg/dl. Il est fourni en version seul ou avec un autopiqueur GlucoJect Dual S, 10 bandelettes réactives GlucoFix iD Sensor et 10 lancettes GlucoJect Lancets.

Mode d'emploi

S'utilise avec les bandelettes GlucoFix iD Sensor.
