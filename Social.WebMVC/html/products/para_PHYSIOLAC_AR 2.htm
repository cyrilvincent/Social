PHYSIOLAC AR 2

Domaines d’utilisation : Alimentation pour enfant - Régurgitations

Formes, Présentations

Poudre instantanée : :

Boîte métal de 900 g (EAN 3518646025235).

Composition

Lait écrémé ; matières grasses végétales ; lactosérum déminéralisé ; amidon de maïs précuit ; maltodextrines ; fibres alimentaires (2 %) : galacto-oligosaccharides, fructo-oligosaccharides ; protéines lactiques ; sels minéraux : phosphate de potassium, carbonate de calcium, chlorure de magnésium, citrate de sodium, phosphate de calcium, gluconate de fer, chlorure de sodium, chlorure de potassium, gluconate de zinc, gluconate de manganèse, gluconate de cuivre, sélénite de sodium, iodure de potassium ; émulsifiant : lécithine de soja ; vitamines : acide L-ascorbique, nicotinamide, acide pantothénique, alpha acétate de tocophérol, acétate de rétinol, riboflavine, pyridoxine, thiamine, acide folique, phytoménadione, cholécalciférol, biotine, cyanocobalamine ; bitartrate de choline ; antioxygènes : DL-alpha-tocophérol, palmitate de L-ascorbyle ; inositol.

Utilisation

Aliment diététique destiné à des fins médicales spéciales en cas de régurgitations pour les nourrissons de 6 à 12 mois. Formule épaissie à l'amidon de maïs.

Mode d'emploi

Se conformer aux instructions figurant sur la boîte.

1 mesurette rase, non tassée, pour 30 ml d'eau.
