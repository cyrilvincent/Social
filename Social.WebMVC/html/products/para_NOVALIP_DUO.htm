NOVALIP DUO

Domaines d’utilisation : Maquillage - Lèvres

Présentation

Stick de 4 ml. Existe en 15 teintes : 05, 11, 22, 35, 40, 60, 66, 73, 158, 170, 173, 184, 185, 191 et 198.

Composition

Isononyl isononanoate, bis-diglyceryl polyacyladipate-2, oleyl erucate, pentaerythrityl tetra isostearate, cera microcristallina/microcrystalline wax, glycerin, polyvinyl laurate, copernicia cerifera cera/carnauba wax, silica dimethyl silylate, panthenol, trioleyl phosphate, BHT (± may contain : CI 77891/titanium dioxide, mica). Code FIL : B47947/1.

Trimethylsiloxyphenyl dimethicone, isohexadecane, Simmondsia chinensis oil/jojoba seed oil, polyethylene, hydrogenated styrene/methyl styrene/indene copolymer, polybutene, hydrogenated polyisobutene, acrylic acid/isobutyl acrylate/isobornyl acrylate copolymer, octyldodecyl neopentanoate, ozokerite, alumina, calcium aluminum borosilicate, silica, calcium sodium borosilicate, synthetic fluorphlogopite, pentaerythrityl tetra-di-t-butyl hydroxyhydrocinnamate, tin oxide, capryloyl salicylic acid (± may contain : CI 77891/titanium dioxide, CI 15850/red 7, CI 77491, CI 77492, CI 77499/iron oxides, CI 15985/yellow 6 lake, CI 77820/silver, CI 19140/yellow 5 lake, CI 42090/blue 1 lake, mica). Code FIL : B47839/1.

Propriétés

Contient un actif microlissant pour uniformiser et adoucir la surface des lèvres. Homogène, la surface des lèvres accroche mieux la couleur pour une tenue longue durée et permet une meilleure réflexion de la lumière, pour un éclat intense de la couleur.

Mode d'emploi

Appliquer sur les lèvres aussi souvent que nécessaire.

ACL : 9952529 (n° 05 : rose pêche) ; 9952392 (n° 11 : mauve douceur) ; 9952446 (n° 22 : cassis festif) ; 9952535 (n° 35 : rose fruité) ; 9952400 (n° 40 : beige nude) ; 9952417 (n° 60 : brun cacao) ; 9952498 (n° 66 : corail indien) ; 9952512 (n° 73 : orange miel) ; 9952541 (n° 158 : cassis nocturne) ; 9952423 (n° 170 : brun sépia) ; 9952452 (n° 173 : brun ombré) ; 9952506 (n° 184 : orange fusion) ; 9952469 (n° 185 : orange laser) ; 9952475 (n° 191 : pur rouge) ; 9952481 (n° 198 : rouge mat).
