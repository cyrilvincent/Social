PHYSIOLAC 3 Croissance

Domaines d’utilisation : Laits de croissance

Formes, Présentations

Poudre instantanée :

Boîte métal de 900 g (EAN 3518646123061).

Composition

Lait écrémé ; matières grasses végétales ; lactosérum déminéralisé ; maltodextrines ; lactose ; fructose ; fibres alimentaires (2 %) : galacto-oligosaccharides, fructo-oligosaccharides ; sels minéraux : carbonate de calcium, phosphate de potassium, chlorure de magnésium, carbonate de potassium, phosphate de sodium, gluconate de fer, chlorure de sodium, citrate de sodium, gluconate de zinc, gluconate de cuivre, sélénite de sodium, gluconate de manganèse, iodure de potassium ; arôme vanille ; vitamines : acide L-ascorbique, nicotinamide, acide pantothénique, alpha acétate de tocophérol, acétate de rétinol, riboflavine, pyridoxine, thiamine, acide folique, phytoménadione, cholécalciférol, biotine, cyanocobalamine ; émulsifiant : lécithine de soja ; acide citrique ; antioxygènes : DL-alpha-tocophérol, palmitate de L-ascorbyle ; inositol ; bêta-carotène.

Utilisation

Aliment lacté en poudre pour les enfants en bas âge de 1 à 3 ans.

Mode d'emploi

Se conformer aux instructions figurant sur la boîte.

1 mesurette rase, non tassée, pour 30 ml d'eau.
