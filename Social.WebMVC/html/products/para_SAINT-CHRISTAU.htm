SAINT-CHRISTAU

Domaines d’utilisation : Thermalisme - Dermatologie et stomatologie

Caractéristiques des eaux

Eaux ferro-cuivriques, bicarbonatées calciques, 13 °C.

Indications

Dermatologie (DER) :

Eczémas secs ou suintants ; affections dermatologiques diverses ; ulcères variqueux, plaies atones, escarres ; séquelles de brûlures ; cicatrices vicieuses, chéloïdiennes ; retard de cicatrisations ; suites de chirurgie esthétique ; lésions chroniques du cuir chevelu ; névrodermites.

Affections de la muqueuse buccolinguale (AMB) :

Glossites ; glossodynie ; langue géographique ; stomatites ; aphtoses ; bimétallisme ; allergie aux prothèses ; leucoplasies ; lichen plan buccal ; parodontites.

Techniques de cure

Cure de boisson ; bains avec eau courante, avec douche sous-marine, local, avec irrigation vaginale ou anale ; douches générale au jet, filiforme, gingivale ; cataplasmes de boue ; humage nébulisation ; pulvérisation ; vaporarium.
