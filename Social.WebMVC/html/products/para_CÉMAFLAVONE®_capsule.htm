CÉMAFLAVONE® capsule

Domaines d’utilisation : Compléments alimentaires - Circulation veineuse

Formes, Présentations

Capsule :

Boîte de 30 (ACL 4756982).

Composition

Huile de tournesol raffinée ; agent d'enrobage : gélatine de poisson ; vitamine C ; eau purifiée ; extrait de citrus ; oxyde de magnésium ; agent d'enrobage : glycérol ; sorbitol 70 % non cristallisé ; huile de soja hydrogénée non OGM ; émulsifiant : lécithine de soja non OGM ; arôme naturel de citron E 123067 ; colorants : titane oxyde E 171, jaune orangé S E 110, jaune de quinoléine E 104.

Composition nutritionnelle pour 1 capsule :

Vitamine C : 150 mg (250 % des AJR). Extrait de citrus : 110 mg. Magnésium : 45 mg (15 % des AJR).

*
AJR : apports journaliers recommandés.

Conseils d'utilisation

1 capsule par jour à avaler le matin, ou selon les recommandations de son médecin. Une cure de 3 mois est conseillée.

Précautions d'emploi

Les compléments alimentaires doivent être utilisés dans le cadre d'une alimentation variée et d'un mode de vie sain, et tenus hors de portée des enfants. Ne pas dépasser la dose recommandée. Conserver dans un endroit sec et frais.
