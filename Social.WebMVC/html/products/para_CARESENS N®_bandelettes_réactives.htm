Présentation, Propriétés

Bandelettes compatibles avec le lecteur CareSens N vendu dans le kit Le lecteur Service + (ACL : 3401020030089, lecteur de renouvellement), le set PAM 2.0 découverte (ACL : 3401020876373), et le set Dinno Tandem (ACL : 3401026407267).

  • Boîte de 100 bandelettes (ACL : 3401096150452).

  • Microgoutte de sang : 0,5 μl.

  • Aspiration par capillarité.

  • Forme triangulaire pour faciliter l'aspiration de la goutte de sang.

  • Film de protection.

Indications

Autosurveillance du diabète.

Mode d'emploi

Utilisation avec le lecteur CareSenS N. Insérer la bandelette, prélever une goutte de sang, consulter le résultat.

Effectuer les tests dans un endroit sec et tempéré, à l'abri des rayons du soleil.
