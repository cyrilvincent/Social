PHYSIOLAC Relais 2

Domaines d’utilisation : Préparations de suite (laits 2e âge)

Formes, Présentations

Poudre instantanée :

Boîte métal de 900 g (EAN 3518646123016).

Composition

Lait écrémé ; lactosérum déminéralisé ; matières grasses végétales ; maltodextrines ; sirop de glucose (polymères de glucose) ; fibres alimentaires (2 %) : galacto-oligosaccharides, fructo-oligosaccharides ; sels minéraux : phosphate de calcium, carbonate de potassium, chlorure de magnésium, carbonate de calcium, phosphate de sodium, chlorure de potassium, phosphate de potassium, gluconate de fer, gluconate de zinc, sélénite de sodium, gluconate de cuivre, gluconate de manganèse, iodure de potassium ; lactose ; émulsifiant : lécithine de soja ; vitamines : acide L-ascorbique, nicotinamide, acide pantothénique, alpha acétate de tocophérol, acétate de rétinol, riboflavine, pyridoxine, thiamine, acide folique, phytoménadione, cholécalciférol, biotine, cyanocobalamine ; bitartrate de choline ; L-tryptophane ; antioxygènes : DL-alpha-tocophérol, palmitate de L-ascorbyle ; inositol.

Utilisation

Lait de suite pour nourrissons de 6 à 12 mois, en relais de l'allaitement ou du lait pour nourrissons.

Mode d'emploi

Se conformer aux instructions figurant sur la boîte.

1 mesurette rase, non tassée, pour 30 ml d'eau.
