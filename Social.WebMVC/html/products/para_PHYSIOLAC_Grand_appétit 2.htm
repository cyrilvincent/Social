PHYSIOLAC Grand appétit 2

Domaines d’utilisation : Préparations de suite (laits 2e âge)

Formes, Présentations

Poudre instantanée :

Boîte métal de 900 g (EAN 3518646123054).

Composition

Lait écrémé ; matières grasses végétales ; maltodextrines ; amidon de maïs précuit ; lactosérum déminéralisé ; fibres alimentaires (2 %) : galacto-oligosaccharides, fructo-oligosaccharides ; protéines lactiques ; sels minéraux : phosphate de potassium, carbonate de calcium, chlorure de magnésium, phosphate de calcium, phosphate de sodium, gluconate de fer, chlorure de potassium, gluconate de zinc, gluconate de cuivre, sélénite de sodium, gluconate de manganèse, iodure de potassium ; lécithine de soja ; vitamines : acide L-ascorbique, nicotinamide, acide pantothénique, alpha acétate de tocophérol, acétate de rétinol, riboflavine, pyridoxine, thiamine, acide folique, phytoménadione, cholécalciférol, biotine, cyanocobalamine ; L-tryptophane ; antioxygènes : DL-alpha-tocophérol, palmitate de L-ascorbyle ; bitartrate de choline ; inositol.

Utilisation

Lait de suite pour nourrissons de 6 à 12 mois au grand appétit.

Mode d'emploi

Se conformer aux instructions figurant sur la boîte.

1 mesurette rase, non tassée, pour 30 ml d'eau.
