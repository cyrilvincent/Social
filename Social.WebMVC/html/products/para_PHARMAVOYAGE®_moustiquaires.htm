PHARMAVOYAGE® moustiquaires

Domaines d’utilisation : Moustiquaires imprégnées, diffuseurs

Formes, Présentations

Moustiquaire imprégnée longue durée (4 ans d'efficacité ou 20 lavages) :

  • Pharmavoyage® Totem : modèle conique 2 personnes, 1 point d'accroche (ACL  9836071 ; EAN 7640144283063).

  • Pharmavoyage® Treck : modèle conique 1 personne, 1 point d'accroche (ACL  9836102 ; EAN 7640144283070).

Composition

Tulle polyester 75 deniers : imprégné de deltaméthrine (55 mg/m2).

Propriétés

  • Protection efficace contre les maladies transmises par les insectes, telles que le paludisme ou la dengue.

  • Peut être utilisé par les nourrissons (recommandé par le Groupe de pédiatrie tropicale).

Conseils d'utilisation

L'utilisation de moustiquaires imprégnées longue durée d'insecticide est recommandée par l'OMS dans tous les pays où sévit le paludisme.

Réimprégnation possible avec Biovectrol® tissu (ACL  7739139 ; EAN 7612013391391).

Renseignements administratifs

Non remb Séc soc.
