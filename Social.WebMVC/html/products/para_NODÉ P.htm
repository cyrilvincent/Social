NODÉ P

Domaines d’utilisation : Shampooings antipelliculaires/États squameux

Formes, Présentations

Shampooing :

Flacons de 400 ml.

Normalisant : ACL 4071020 ; Apaisant : ACL 4071008 ; Purifiant : ACL 4071043 ; Restructurant : ACL 4071072 ; Volumateur : ACL 4071103.

Composition

Noyau d'actifs antipelliculaire commun : piroctone olamine, climbazole, acide salicylique.

Cuirs chevelus irrités/Apaisant : D-panthénol ; Cheveux gras/Purifiant : gluconate de zinc ; Cheveux secs/Restructurant : huile d'amande douce ; Cheveux fins/Volumateur : conditionneurs.

Propriétés

Nettoie en douceur, élimine biologiquement les pellicules et évite leur réapparition. Apaise les démangeaisons, assainit le cuir chevelu et apporte brillance et souplesse aux cheveux.

Utilisation

Pellicules modérées récidivantes ou occasionnelles.

Mode d'emploi

  • Période d'attaque (3 semaines) : 3 shampooings par semaine.

  • Entretien : 1 à 2 shampooings par semaine.
