ALPHA KM Soin anti-âge correcteur rides installées

Domaines d’utilisation : Soins du visage - Vieillissement cutané

Formes, Présentations

Crème :

Flacon-pompe de 30 ml (ACL 4831175).

Composition

Biocéramides VI ; AHA ; lactate d'ammonium.

Propriétés, Utilisation

Anti-âge kérato-modulateur composé d'AHA et de biocéramides VI qui lissent et restructurent. Contient du lactate d'ammonium qui régénère la peau.

Mode d'emploi

Appliquer le matin sur une peau préalablement nettoyée. Parfaite base de maquillage.
